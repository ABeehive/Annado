# Delano Context Pack

This folder is the durable project memory for Annado. Keep it factual, compact, and aligned with
current repo state. It should help a new agent resume work without rereading the entire codebase.

Read order for most work:

1. `project-overview.md`
2. `project-brief.md`
3. `tech-context.md`
4. The task-specific file: structure, system patterns, product context, style, progress, or GUI
   testing

`.project/projects/` remains the execution source of truth for active Delano projects. This context
pack summarizes reality; it does not replace specs, plans, tasks, or evidence.
